h1. My vimfiles.

h2. To download:
<pre><code>
git clone git://github.com/eletronzip/vimfiles.git ~/.vim
</pre></code>

h2. To install the bundled plugins:
<pre><code>
git submodule update --init
</pre></code>

h2. To source the vimrc:
<pre><code>
echo "source ~/.vim/vimrc" > ~/.vimrc
</code></pre>
